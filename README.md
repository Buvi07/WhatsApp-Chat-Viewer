# 📱 WhatsApp Chat Viewer

A simple and interactive web application to **view WhatsApp exported chats** in a clean UI with media support and PDF export functionality.

---

## 🚀 Features

* 📂 Upload WhatsApp chat `.txt` file
* 🖼️ Load media files via `.zip`
* 💬 WhatsApp-style chat interface
* 🧾 Supports multi-line messages
* 🖼️ Displays images and videos
* 📅 Date-wise message grouping
* 🔐 Safe rendering (XSS protected)
* 📄 Export chat to PDF
* ⚡ Smooth scrolling and responsive design

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Bootstrap 5)
* JavaScript (Vanilla JS)
* JSZip (for extracting media files)
* html2pdf.js (for PDF export)

---

## 📂 How to Use

1. Export your WhatsApp chat:

   * Open chat → Tap **3 dots** → *More* → *Export Chat*
   * Choose **Without Media** or **Include Media**

2. Open the project in your browser

3. Upload files:

   * Select `.txt` chat file
   * (Optional) Upload `.zip` media file

4. Click **View Chat**

5. Click **Export as PDF** (optional)

## ⚠️ Limitations

* Supports standard WhatsApp export format
* Very large chats may take time to render or export
* Videos in PDF are shown as preview (not playable)
